## CommitRule

```
1. 在CodeRecord中创建自己名字的目录——以后每天的刷题都提交到该目录
2. 包名可以自己定义
3. 每日刷题的类名以 Day_年月日_LeetCode题目编号 (年只取后两位,月日补齐4位) 
   例如:2022年4月1号刷409题 类名:Day_220401_409）
4. 每日提交后在微信群 发布github提交截图 LeetCode提交记录截图
```
